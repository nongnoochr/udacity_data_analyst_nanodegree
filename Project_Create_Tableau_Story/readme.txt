The dataset in '2008_master_7_28_18.csv' is used in this project.

This is the 'Flights' data of 2008 the is provided in the Data Set Options and the original dataset was downloaded from 

http://stat-computing.org/dataexpo/2009/the-data.html

Note that the original set was cleaned in Python and the cleaned data set was saved to '2008_master_7_28_18.csv' which is the data set that is used in this project.
